# ChunkSizeSweetSpot

D3 line chart showing retrieval quality as a function of chunk size on a log scale (32–4096 chars). The inverted-U curve peaks in the 200–800 char sweet spot, with shaded zones indicating fragmentation (red, too small) and context dilution (amber, too large).

## Interactions
- Drag the slider to select any chunk size; marker moves on the curve and a status badge shows the current quality score and sweet-spot status.

## Placement
At the end of "The tradeoff, summarized" section in Chapter 1, right after the existing comparison table.
