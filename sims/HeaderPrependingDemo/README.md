# HeaderPrependingDemo

Side-by-side comparison of a chunk with and without a header prefix prepended. Shows simulated cosine similarity scores and retrieval outcomes for the query "what is the return window?" — demonstrates concretely why prepending headers drastically improves retrieval.

## Interactions
- Toggle "Show LLM input/output" to see what a downstream LLM would do with each retrieved chunk

## Placement
In the "The single highest-leverage chunking trick" sub-section of Chapter 1, after the descriptive prose.
