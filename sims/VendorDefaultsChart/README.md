# VendorDefaultsChart

Horizontal bar chart showing chunk-size defaults from 5 real vendors/papers. Illustrates the 10x spread in production system defaults, reinforcing that chunk size is a corpus-specific hyperparameter, not a universal constant.

No interactivity — static D3 SVG rendered inline in the chapter prose.

## Placement
After the bulleted list of vendor defaults in the "Numbers from real systems" sub-section of Chapter 1.
